Settings for VSCode
===

## Requirement
- Node.js v6.11.0
- eslint v4.1.1

## Usage
```
# Install Extension for VSCode
./installExtension.sh
```

## Extension
- [Vim](https://marketplace.visualstudio.com/items?itemName=vscodevim.vim)
- [HTMLHint](https://marketplace.visualstudio.com/items?itemName=mkaufman.HTMLHint)
